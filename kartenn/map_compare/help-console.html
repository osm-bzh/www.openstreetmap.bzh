Map Compare Admin Console

The Map Compare Admin Console allows you to configure and store the default number of maps, the default center of maps, and the order of maps.

It is also possible to configure your own OpenStreetMap tile server by a standard OSM tile URL. You can configure up to 3 servers. This will work even on a local server in an intranet.

The Map Compare admin console is for experts only. Use it on your on own risk!!! All values will be stored in a persistent cookie. If you delete all your cookies in the browser, all your Map Compare preferences will be lost.

For more information about tile servers, please read the OpenStreetMap Wiki at https://wiki.openstreetmap.org/wiki/TMS
